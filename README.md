# Jogo de nave - Bootcamp DIO

Projeto de um jogo de nave utilizando o HTML5 juntamente com as folhas de estilo CSS3 e o JavaScript, é possível desenvolver jogos de forma rápida e compatível com diversas plataformas, incluindo dispositivos móveis. No projeto são discutidas as etapas de desenvolvimento de jogos em HTML5, CSS3 e JavaScript do início ao fim, utilizando a didática passo a passo em seu desenvolvimento. 


## Motivação

Projeto para fins de estudo, aprendizagem e prática em JavaScript, CSS e HTML.


## Tecnologias utilizadas

- JavaScript
- jQuery
- HTML
- CSS
- Visual Studio Code
